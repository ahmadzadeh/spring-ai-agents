# Supplier Info Fetch Skill

## How to search effectively

1. Start with the supplier's official domain if known. Otherwise search: `"<Supplier Name>" official site`
2. For certifications, search: `"<Supplier Name>" ISO 27001 certificate` or `"<Supplier Name>" SOC 2`
3. For adverse info: `"<Supplier Name>" lawsuit OR fine OR sanction OR breach`
4. For company registry: search `"<Supplier Name>" company number <country>` or use national registries (e.g. Companies House for UK)
5. For ESG: `"<Supplier Name>" sustainability report OR CSR report`

## Search quality rules

- Prefer primary sources (official company pages, government registries, audit bodies) over aggregators.
- For each piece of evidence, capture the URL and a brief description of what the page contains.
- If a PDF is found (e.g. ISO certificate, annual report), note the URL even if you cannot read the full content.
- Search in both English and the supplier's local language if known.

## Completeness checklist

Before finishing, confirm you have searched for:
- [ ] Company website and legal identity
- [ ] LinkedIn / professional network presence
- [ ] Customer review platforms
- [ ] Certifications (quality, security, environmental)
- [ ] News / press mentions (last 2 years)
- [ ] Sanctions / adverse media
- [ ] Financial information (annual reports, credit mentions)
